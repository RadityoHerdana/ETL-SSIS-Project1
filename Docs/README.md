Welcome to the ETL-SSIS-Project1 repository!

Repository ini dibuat sebagai dokumentasi sekaligus portfolio implementasi proses Extract, Transform, Load (ETL) menggunakan SQL Server Integration Services (SSIS). Project ini mencakup seluruh tahapan ETL, mulai dari pengolahan data mentah, proses transformasi, hingga pemuatan data ke SQL Server.

Contents

1. Docs
Folder ini berisi dokumentasi proyek, diagram arsitektur, alur ETL, serta penjelasan implementasi yang digunakan selama pengembangan.

2. Resources
Folder ini berisi seluruh file pendukung yang digunakan dalam proyek, seperti dataset contoh, dokumen referensi, dan file pendukung lainnya.

3. Source-Data
Folder ini berisi data mentah (raw data) yang digunakan sebagai sumber data sebelum diproses pada tahap ETL.

4. SQL
Folder ini berisi seluruh script SQL yang digunakan dalam proyek, seperti pembuatan database, tabel, stored procedure, serta query untuk validasi hasil ETL.

5. SSIS
Folder ini berisi project SQL Server Integration Services (SSIS), termasuk solution, project, dan package (.dtsx) yang digunakan untuk menjalankan proses ETL.

6. Output
Folder ini berisi hasil proses ETL, seperti data yang telah dimuat ke database, file output, maupun log hasil eksekusi.

7. Screenshots
Folder ini berisi dokumentasi berupa screenshot Control Flow, Data Flow, hasil query SQL Server, serta hasil implementasi ETL.

Repository ini akan terus diperbarui dengan penambahan studi kasus ETL, pengembangan package SSIS, serta penerapan best practice yang umum digunakan dalam proyek Data Engineering.
